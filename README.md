# 爱乐在线音乐网站

## 项目简介
爱乐在线音乐网站是在线音乐网站，在网站可以在线播放歌曲，以歌单、歌手等维度查找对应的歌曲，整个系统分为在线听歌的PC端和后台管理系统。适合毕业设计、大作业、课程设计等

## 下载地址
https://bysj.bsitem.com/#/home/goodDetail?goodId=1074

## 技术介绍
Java;SpringBoot;Vue;ElementUI;Mysql

## 开发工具
1.IntelliJ IDEA或者Myeclipse 3.Navicat 3.Mysql 4.JDK8

## 项目资料包含
1. 前后端源代码
2. 开题报告（任务书）+ 配套论文（一万两千字以上）+ 答辩PPT
3. 远程控屏调试（向日葵或者toDesk控屏）
4. 一键启动项目（双击运行演示，无需搭建复杂环境）
5. 项目讲解视频教程
